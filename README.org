* About

This repository contains the source files for [[https://lisp.netlify.app][Hiroshima.lisp]], a blog built with Hugo.

* Getting Started

** Creating a New Article

Generate a new blog post from the template:

#+begin_src shell
hugo new blog/{{blog-title}}
#+end_src

** Running the Development Server

Preview your changes locally with live reload:

#+begin_src shell
hugo server -D
#+end_src

** Deploying

The blog is automatically deployed to [[https://lisp.netlify.app][lisp.netlify.app]] via Netlify when changes are pushed to the master branch:

#+begin_src shell
git push origin master
#+end_src
